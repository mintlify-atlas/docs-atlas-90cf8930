# Uploading Documents

My Auto Guide provides secure cloud storage for all your important vehicle documents. Keep digital copies of your SOAT, Tecnomecánica, insurance, and ownership documents accessible anywhere.

## Document Categories

The app organizes documents into four categories:

1. **SOAT** - Mandatory vehicle insurance
2. **Tecno** - Tecnomecánica (technical inspection certificate)
3. **Seguro** - Additional insurance policies
4. **T. Propied** - Tarjeta de Propiedad (vehicle ownership card)

Each category can store multiple files, giving you a complete digital vehicle file.

<Note>
All documents are stored securely in the cloud and linked to your account. They remain private and accessible only to you.
</Note>

## Supported File Types

You can upload the following file formats:

- **PDF** documents (.pdf)
- **Images**: JPEG (.jpg, .jpeg), PNG (.png), WebP (.webp), HEIC (.heic), GIF (.gif)

<Tip>
PDF is recommended for official documents as it preserves formatting and is universally compatible. Photos work great for quick captures of physical documents.
</Tip>

## Uploading Documents

<Steps>

### Step 1: Access Document Manager

From your vehicle dashboard:
1. Scroll to the **"Documentos"** section
2. You'll see four document cards: SOAT, Tecno, Seguro, and T. Propied
3. Tap on the document type you want to manage

### Step 2: Open the Document Gallery

When you tap a document card:
- A modal sheet slides up from the bottom
- The title shows "Documentos: [Document Type]"
- Any existing files are displayed in a grid
- An **"Agregar archivo"** (Add File) button appears at the top

### Step 3: Choose Your Upload Method

Tap **"Agregar archivo"** to see upload options:

**Take a Photo**
- Uses your device camera
- Perfect for capturing physical documents
- Photo is automatically saved and uploaded

**Select from Gallery**
- Browse your device's photo library
- Choose existing photos or scanned documents
- Supports selecting PDFs from file managers

<Warning>
Ensure documents are clear and readable before uploading. Blurry or poorly lit photos may be difficult to reference later.
</Warning>

### Step 4: Upload the File

1. After selecting a file or taking a photo:
   - A loading indicator appears briefly
   - The file uploads to secure cloud storage
   - A thumbnail appears in the document grid

2. The uploaded file is automatically:
   - Timestamped with upload time
   - Compressed for efficient storage
   - Linked to your vehicle record

<Note>
Files are uploaded with unique filenames (timestamp-based) to prevent conflicts if you upload multiple versions of the same document.
</Note>

</Steps>

## Viewing Documents

### In-App Viewing

**For Images:**
1. Tap any image thumbnail in the document grid
2. The image opens in a full-screen viewer
3. Pinch to zoom in/out
4. Swipe to dismiss

**For PDFs:**
1. Tap any PDF thumbnail
2. Opens in the integrated PDF viewer
3. Scroll through pages
4. Zoom for better readability
5. Use back button to return to grid

### External Viewing

For better viewing or sharing:
1. **Long-press** on any document thumbnail
2. The file opens in your device's default app (Photos, PDF reader, etc.)
3. From there you can share, print, or edit as needed

<Tip>
Use long-press to quickly share documents with mechanics, insurance agents, or authorities without leaving the app.
</Tip>

## Organizing Multiple Documents

Each document category can store unlimited files. Best practices:

### Naming Strategy
Files are automatically named with timestamps, but you can identify them by:
- Upload date (shown in filename)
- Thumbnail preview
- File type icon (PDF vs image)

### Version Control
- Keep the most recent version uploaded
- Archive old versions by category
- Delete expired documents to keep your gallery clean

### Recommended Organization

**SOAT Category:**
- Current year's SOAT certificate
- Previous year's certificate (for reference)
- Proof of payment receipts

**Tecno Category:**
- Current inspection certificate
- Previous inspection reports
- Any repair receipts related to inspections

**Seguro Category:**
- Insurance policy documents
- Payment receipts
- Claim forms or correspondence

**T. Propied Category:**
- Front of ownership card
- Back of ownership card
- Transfer documents if applicable

## Deleting Documents

<Steps>

### Step 1: Locate the Document

1. Open the document category
2. Find the file you want to delete in the grid
3. Look for the small delete icon in the top-right corner of each thumbnail

### Step 2: Confirm Deletion

1. Tap the **delete icon** (trash can)
2. A confirmation dialog appears
3. The dialog shows the filename
4. Tap **"Eliminar"** to confirm, or **"Cancelar"** to cancel

### Step 3: File Removed

- The file is immediately deleted from cloud storage
- The thumbnail disappears from the grid
- If it was the assigned document for the category, the card updates to show "upload" state

</Steps>

<Warning>
Deletion is permanent and cannot be undone. Make sure you have other copies of important documents before deleting.
</Warning>

## Document Security

Your documents are protected through multiple layers:

1. **Encrypted Storage** - Files stored in secure cloud buckets
2. **Signed URLs** - Temporary access links expire after 1 hour
3. **User Authentication** - Only accessible when logged into your account
4. **Private Folders** - Each vehicle has its own isolated storage

<Note>
Signed URLs are automatically refreshed when viewing documents, ensuring continuous access while maintaining security.
</Note>

## Tips for Best Results

### Photography Tips

**Lighting**
- Use natural daylight when possible
- Avoid shadows across the document
- Ensure even lighting without glare

**Positioning**
- Lay document flat on a contrasting surface
- Hold camera directly above, parallel to document
- Fill the frame with the document
- Keep all text and edges visible

**Focus**
- Tap the screen to focus on text
- Ensure all text is sharp and readable
- Take multiple shots if needed, upload the best one

### File Size Considerations

- Images are automatically compressed for efficient upload
- PDF files upload at original size
- All files count toward your account storage limit
- Delete old documents periodically to free up space

### When to Update Documents

**Immediate Updates:**
- When you renew SOAT or Tecnomecánica
- After any insurance changes
- If ownership changes

**Regular Reviews:**
- Check quarterly for expired documents
- Update after major repairs or modifications
- Refresh photos if originals become unclear

## Troubleshooting

**Upload fails repeatedly?**
- Check your internet connection
- Ensure the file size isn't excessive (under 10MB recommended)
- Try a different file format (PDF instead of image, or vice versa)

**Can't see uploaded documents?**
- Pull down to refresh the document gallery
- Check you're viewing the correct document category
- Ensure you're logged into the correct account

**Document viewer won't open?**
- For PDFs, ensure the file wasn't corrupted during upload
- For images, try long-pressing to open in external app
- Re-upload the document if issues persist

**Delete button not working?**
- Ensure you have an active internet connection
- Try closing and reopening the document manager
- Check that you're the owner of the document
